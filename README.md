# Solstice Opal Ancillary Revenue Analysis

This project analyzes ancillary revenue at Solstice Opal Hotel across spa, dining, and activities. The goal is to identify which guest segments generate the most ancillary revenue and provide recommendations leadership can use to improve revenue.

## Project Files

- `notebook.ipynb` - main DataLab notebook with data cleaning, validation, analysis, and recommendations
- `da_sample_guest_profiles.csv` - guest profile data
- `da_sample_stay_details.csv` - hotel stay details
- `da_sample_ancillary_spend.csv` - ancillary spend transactions
- `chart.png` - exported project chart

## Key Questions

- Which loyalty tiers generate the highest ancillary revenue?
- How does spending differ between business and leisure guests?
- Which booking channels are associated with higher ancillary spend?
- Which ancillary categories generate the most revenue?

## Tools Used

- Python
- pandas
- NumPy
- Matplotlib

## How to Run

Open `notebook.ipynb` in Jupyter Notebook, DataLab, VS Code, or GitHub's notebook viewer. The CSV files should stay in the same folder as the notebook.

To run the notebook locally, install the required packages:

```bash
pip install -r requirements.txt
```
