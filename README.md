# Ancillary-Revenue-Analysis-
**Ancillary Revenue Analysis - Solstice Opal Hotel** 

The purpose of this analysis is to identify which guest segments generate the most ancillary revenue (spa, dining, and activities) at Solstice Opal Hotel, and to provide actionable recommendations that leadership can use to maximise this revenue stream. The analysis draws on three datasets: guest profiles, stay details, and ancillary spend records.
